# 1019  最大子段和

## 问题描述
- N个整数组成的序列a[1],a[2],a[3],…,a[n]，求该序列如a[i]+a[i+1]+…+a[j]的连续子段和的最大值。当所给的整数均为负数时和为0。</br>例如：-2,11,-4,13,-5,-2，和最大的子段为：11,-4,13。和为20。
- Input</br>
第1行：整数序列的长度N（2 <= N <= 50000)</br>
第2 - N + 1行：N个整数（-10^9 <= A[i] <= 10^9）
- Output</br>
输出计算结果，求模结果。

## 思想：二分法，动态规划

- 将序列分为两部分L和R，则序列的最大子段和为L的最大子段和，或R的最大子段和，或经过中间子段的某个子段和。
- 从头开始向后遍历，如果sum<0,则置为零（小于0的前序列怎么加都拖后腿），sum不断与max同步，若sum>max则更新max，否则可得到最大子段和。

## 出现的问题
- 二分法使用的不够熟练，二分的边界，以及如何比较获取最大子段和。
